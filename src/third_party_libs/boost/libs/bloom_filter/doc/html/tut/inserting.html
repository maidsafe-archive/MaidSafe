Inserting Elements

Instantiation is the hardest part about using a Bloom filter. Once that step is complete, insertion, querying, and removal proceed smoothly. The example below shows how to insert an element into the Bloom filter:

basic_bloom_filter<int, 1000> bloom; bloom.insert(1); bloom.insert(std::string("x")); // compile-time error bloom.insert(1.0); // fine: implicit cast float -> int

Range insertion is also supported for all Bloom filter types.
