Intersections and Unions

As stated previously, the union of two Bloom filters is the set of elements that occur in either Bloom filter. The intersection of two Bloom filters is the set of elements that occur in both Bloom filters. The examples below demonstrate how to use this:

basic_bloom_filter<int, 1000> bloom1; basic_bloom_filter<int, 1000> bloom2; basic_bloom_filter<int, 1000> bloom_union; basic_bloom_filter<int, 1000> bloom_intersect; bloom1.insert(1); bloom1.insert(2); bloom2.insert(2); bloom2.insert(3); bloom_union = bloom1 | bloom2; bloom_intersect = bloom1 & bloom2; bloom_union.probably_contains(1); // true bloom_union.probably_contains(2); // true bloom_union.probably_contains(3); // true bloom_intersect.probably_contains(1); // false bloom_intersect.probably_contains(2); // true bloom_intersect.probably_contains(3); // false
