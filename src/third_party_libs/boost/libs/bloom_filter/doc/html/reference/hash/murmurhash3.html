Murmurhash3 Hasher

template <typename T, size_t Size = 0, bool Use128Mode = 0> struct murmurhash3 { size_t operator()(const T&) const; };

A Hasher that uses the public domain murmurhash3 under the covers. This hash function excels where speed is the desired goal. The Seed parameter is merely added to the result. The Use128Mode template parameter determines whether the 128-bit hash is calculated or not. This mode is theoretically fastest in 64-bit mode. Only the least significant (32/64) bits are used in the case of 128-bit mode.
